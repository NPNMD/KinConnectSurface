## üìê Usage Guidelines

### When to Use Each Version

**Primary Logo (Full Color):**
- ‚úÖ Website headers and navigation
- ‚úÖ Marketing materials and brochures
- ‚úÖ Presentations and pitch decks
- ‚úÖ Social media posts with sufficient space
- ‚úÖ Email headers and newsletters

**Icon Only:**
- ‚úÖ App icons and favicons
- ‚úÖ Social media profile pictures
- ‚úÖ Small digital applications
- ‚úÖ When logo is already established in context
- ‚úÖ Navigation elements

**Wordmark Only:**
- ‚úÖ Text-heavy documents
- ‚úÖ Email signatures
- ‚úÖ Footer text
- ‚úÖ Inline with body text
- ‚úÖ When space is limited horizontally

**Stacked Version:**
- ‚úÖ Square social media profiles (LinkedIn, Twitter)
- ‚úÖ App icons with text
- ‚úÖ Vertical layouts and narrow spaces
- ‚úÖ Print materials with square format

**White Version:**
- ‚úÖ Dark backgrounds (navy, black, dark blue)
- ‚úÖ Photography with dark overlays
- ‚úÖ Dark mode interfaces
- ‚úÖ Reversed applications

**Black Version:**
- ‚úÖ Black and white printing
- ‚úÖ Photocopies and fax
- ‚úÖ High contrast applications
- ‚úÖ Newspaper advertisements

**Monochrome Version:**
- ‚úÖ Single-color printing
- ‚úÖ Branded merchandise (single color)
- ‚úÖ Simplified applications
- ‚úÖ Cost-effective printing

### Clear Space Requirements

**Minimum Clear Space:** Equal to the height of one circle on all sides
- No text, graphics, or other logos within clear space
- Maintain clear space even when logo is at minimum size
- Isolation zone protects logo integrity and ensures visibility

### Minimum Sizes

**Digital Applications:**
- Icon only: 24px height minimum
- Full logo: 120px width minimum
- Stacked: 100px width minimum
- Favicon: 16px, 32px, 48px

**Print Applications:**
- Icon only: 0.5 inch height minimum
- Full logo: 2 inches width minimum
- Stacked: 1.5 inches width minimum

**Below Minimum:** Use simplified version without heart detail

## üö´ What NOT to Do

‚ùå **Never:**
- Stretch or distort the logo
- Rotate the logo at angles
- Change the logo colors (except approved variations)
- Add effects (shadows, gradients, outlines, glows)
- Place on busy backgrounds without sufficient contrast
- Recreate or redraw the logo
- Separate icon from wordmark in unapproved ways
- Use low-resolution or pixelated versions
- Alter the spacing between elements
- Change the font or typography

## üì§ Exporting to Other Formats

### Converting SVG to PNG

**For Web Use (72 DPI):**
```bash
# Using Inkscape (command line)
inkscape fammedicalcare-logo-primary.svg --export-type=png --export-dpi=72 --export-filename=fammedicalcare-logo-primary.png

# Using ImageMagick
convert -density 72 fammedicalcare-logo-primary.svg fammedicalcare-logo-primary.png
```

**For Print Use (300 DPI):**
```bash
# Using Inkscape
inkscape fammedicalcare-logo-primary.svg --export-type=png --export-dpi=300 --export-filename=fammedicalcare-logo-primary-print.png

# Using ImageMagick
convert -density 300 fammedicalcare-logo-primary.svg fammedicalcare-logo-primary-print.png
```

### Converting SVG to JPG

```bash
# Using ImageMagick (with white background)
convert -density 300 fammedicalcare-logo-primary.svg -background white -flatten fammedicalcare-logo-primary.jpg
```
